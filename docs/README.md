## 🔧 Key Features

### Intelligence System
- **Semantic Memory**: ChromaDB vector store for contextual search
- **Pattern Learning**: Learns your eating habits and preferences
- **Trend Detection**: Identifies nutrition trends over time
- **Smart Suggestions**: Personalized recommendations based on your data

### Data Enrichment
- **USDA FoodData Central**: 400K+ foods with detailed nutrition
- **OpenFoodFacts**: International product database
- **Indian Foods**: Curated database of common Indian dishes
- **Confidence Scoring**: Multi-factor accuracy assessment

### Core Features
- **Natural Language Logging**: Just describe what you ate
- **Obsidian Integration**: Markdown-first, vault-native
- **Git Sync**: Automatic version control
- **Privacy-First**: All data stays local
